# BedrockProxy
A proxy hack utilities for Minecraft: Bedrock Edition

## Download
- [Releases](https://github.com/AkmalFairuz/BedrockProxy/releases/)

## Run Proxy
```
java -jar BedrockProxy-1.0-SNAPSHOT.jar
```
